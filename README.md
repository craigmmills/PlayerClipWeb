# PlayerClipWeb

Static one-pager for [PlayerClip](https://github.com/craigmmills/PlayerClip) — splash homepage, privacy policy, and support page.

Hosted on GitHub Pages at: https://craigmmills.github.io/PlayerClipWeb/

## Pages

- `index.html` — splash / coming-soon
- `privacy.html` — privacy policy (linked from App Store Connect)
- `support.html` — support page (linked from App Store Connect)

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

Pushes to `main` auto-deploy via GitHub Pages. No build step.

## Branding

Matches PlayerClip's "Broadcast Volt" design system:

- Background `#0A0B0D`
- Accent `#D7FF3E` (volt yellow-green)
- Type: Barlow Condensed (Google Fonts)
